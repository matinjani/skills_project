# Resume Builder

Transform your experience into a professional, internship-ready resume.

## Input Gathering

Collect information in this order:

### Step 1: Basic Info
- Full name
- Email
- Phone
- LinkedIn (optional)
- GitHub/Portfolio (optional for tech)
- Location (City, Country)

### Step 2: Education
- University/College name
- Degree and Major
- Expected graduation (Month Year)
- GPA (if above 3.0 / 7.0 / 70%)
- Relevant coursework (optional)

### Step 3: Experience
For each experience, ask:
- Company/Organization name
- Role/Title
- Dates (Month Year - Month Year)
- 2-4 things you did (will convert to bullet points)

Types to include:
- Internships
- Part-time jobs
- Freelance work
- Teaching/Tutoring

### Step 4: Projects
For each project:
- Project name
- Brief description
- Technologies/tools used
- Your role (if team project)
- Link (if available)

### Step 5: Skills
- Technical skills (programming, tools, software)
- Languages
- Soft skills (limit to 3-4 most relevant)

### Step 6: Extras (Optional)
- Certifications
- Awards/Achievements
- Volunteering
- Leadership roles

## Bullet Point Formula

Transform weak bullets into strong ones using **XYZ Formula**:

**Accomplished [X] as measured by [Y] by doing [Z]**

### Before → After Examples

| Weak | Strong |
|------|--------|
| "Worked on website" | "Developed responsive website using React, improving load time by 40%" |
| "Helped customers" | "Resolved 50+ customer inquiries daily, maintaining 95% satisfaction rate" |
| "Did data entry" | "Processed 200+ records weekly with 99% accuracy using Excel automation" |
| "Made social media posts" | "Created social media content reaching 10K+ followers, increasing engagement by 25%" |

### Action Verbs by Category

**Technical:** Developed, Built, Implemented, Automated, Designed, Engineered, Programmed, Optimized

**Leadership:** Led, Managed, Coordinated, Directed, Supervised, Mentored, Organized

**Achievement:** Achieved, Increased, Reduced, Improved, Exceeded, Delivered, Launched

**Analysis:** Analyzed, Researched, Evaluated, Assessed, Identified, Investigated

## Tailoring for Different Roles

### Tech/Software Internship
- Emphasize: Projects, GitHub, Technical skills
- Include: Programming languages, frameworks, tools
- Projects section before/equal to Experience

### Business/Marketing Internship
- Emphasize: Leadership, Communication, Results
- Include: Campaigns, metrics, team collaboration
- Experience section first

### Research Position
- Emphasize: Academic projects, Publications, Lab work
- Include: Research methods, papers, presentations
- Education section expanded

## Quality Checklist

Before finalizing:
- [ ] One page only (for students)
- [ ] No typos or grammar errors
- [ ] Consistent formatting (dates, bullets, spacing)
- [ ] Action verbs start each bullet
- [ ] Quantified results where possible
- [ ] Contact info complete and professional
- [ ] Most relevant content first
- [ ] No personal info (age, photo, marital status)
